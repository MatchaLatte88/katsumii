Compatibility

Runs on Windows and macOS.

The core journal, analytics, reports, backups, manual imports, and supported API integrations are available across supported desktop platforms.

Runs on

  • Windows 10 or newer
  • Windows 11 recommended
  • macOS 12 Monterey or newer
  • Intel/AMD 64-bit and Apple Silicon

System requirements

  • 8 GB RAM
  • 1 GB free storage
  • 1440 x 900 display or larger
  • Internet connection for product activation, updates, API sync, and integrations
Integration note

MetaTrader 5 sync is currently Windows-only and requires a locally installed, running MT5 terminal. On macOS, Katsumii supports manual imports, analytics, reports, backups, and supported API integrations.

API access

API usage may be paid and depends on your broker. Katsumii works with broker integrations only when you have a matching API key.
